错误处理

有时候，你有一个复杂的应用程序，你不知道有多少翻译。你也不知道是否有一个漏译特定ID。这是漏译的处理程序进来。

使用日志处理程序扩展失踪翻译

有一个针对angular-translate的扩展，如果你试着翻译不存在翻译的ID他将日志记录传到控制台。像所有的扩展一样，您可以通过 Bower 这样安装：

$ bower install angular-translate-handler-log

要使用它你只需要在$translateProvider上调用useMissingTranslationHandlerLog(). angular-translate 会帮你完成其他所需。

$translateProvider.useMissingTranslationHandlerLog();
